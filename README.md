# VSCode Theme

Build tools for the **Wildpeaks VSCode Theme**.

Install the extension from the Marketplace: [wildpeaks.wildpeaks-theme](https://marketplace.visualstudio.com/items?itemName=wildpeaks.wildpeaks-theme).
